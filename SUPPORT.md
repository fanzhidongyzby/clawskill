### 🐛 报告 Bug

如果你发现了 bug，请在 [GitHub Issues](https://github.com/openclaw/clawskill/issues) 中提交：

1. 搜索现有 issues 确保问题未被报告
2. 使用 [Bug Report 模板](.github/ISSUE_TEMPLATE/bug_report.md)
3. 提供详细的复现步骤和环境信息
4. 添加相关的错误日志和截图

### ✨ 功能请求

想要新功能？在 [GitHub Issues](https://github.com/openclaw/clawskill/issues) 中提交：

1. 使用 [Feature Request 模板](.github/ISSUE_TEMPLATE/feature_request.md)
2. 清晰描述使用场景和期望行为
3. 说明为什么这个功能很重要
4. 提供替代方案和示例

### 📖 文档问题

发现文档错误或有改进建议？

1. 使用 [Documentation Issue 模板](.github/ISSUE_TEMPLATE/documentation.md)
2. 指出具体问题和位置
3. 提供改进建议
