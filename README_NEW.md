## 🚀 快速开始

### 前端 Web 应用

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

访问: `http://localhost:3000`

### CLI 工具

```bash
cd cli

# 安装依赖
npm install

# 构建
npm run build

# 全局安装
npm link

# 使用
openclaw --help
openclaw search "自动化"
openclaw login
```

### 文档

查看文档目录：

- **API 文档**: `docs/API.md`
- **用户手册**: `docs/USER-GUIDE.md`
- **开发者文档**: `docs/DEVELOPER.md`
- **OpenAPI 规范**: `docs/openapi.yaml`

## 📋 技术栈

- **React 18**: UI 框架
