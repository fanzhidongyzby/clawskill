# ClawSkill 待办事项（基于 EvoMap 关系）

---

## 一、重新定位

### 核心使命

**为智能体和人类提供统一的技能发现、检索和安装入口，并成为 EvoMap 的技能源和前端门户**

**价值主张：**
1. **统一技能门户** - 聚合 GitHub、NPM、PyPI 等所有技能源
2. **机器可读** - AI Agent 自动发现、检索和安装技能
3. **人类可读** - 开发者浏览、搜索、管理和贡献技能
4. **EvoMap 前端** - 展示 EvoMap 验证的 Capsule，提供一键安装
5. **技能源输入** - 为 EvoMap 提供原始技能包进行实验验证

### 与 EvoMap 的关系

| 维度 | EvoMap | ClawSkill |
|------|--------|-----------|
| **核心价值** | 实验验证 + 进化资产 | 技能聚合 + 统一门户 |
| **输入** | 技能包（原始代码） | GitHub/NPM/PyPI 等 |
| **输出** | Capsule（验证资产） | 统一技能索引 + 安装 API |
| **验证机制** | 大量实验 + 概率复现 | 基础扫描 + 用户反馈 |
| **自动化程度** | 完全自动（AI native） | 半自动 + 人工辅助 |
| **目标用户** | AI Agent（主要） | AI Agent + 人类开发者 |

### 互补协同模式

```
┌─────────────────────────────────────────────────────────────┐
│                    协同工作流                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   GitHub/NPM/PyPI  ──┐                                     │
│        │             │                                     │
│        ▼             │                                     │
│   ┌─────────┐   技能源适配器   ┌─────────────┐             │
│   │原始技能 │  ──────────────> │ ClawSkill   │             │
│   └─────────┘                  │  统一门户   │             │
│                                 └──────┬──────┘             │
│                                        │                    │
│                                        │ A2A 协议          │
│                                        ▼                    │
│                                 ┌─────────────┐             │
│                                 │  EvoMap     │             │
│                                 │ 实验验证平台 │             │
│                                 └──────┬──────┘             │
│                                        │                    │
│                                        │ Capsule API       │
│                                        ▼                    │
│                                 ┌─────────────┐             │
│                                 │   Capsule   │             │
│                                 │  (验证资产) │             │
│                                 └──────┬──────┘             │
│                                        │                    │
│                                        │ 展示 + 推广        │
│                                        ▼                    │
│   AI Agent ──────────────────────────>  ClawSkill          │
│   (自动安装、使用)                      (展示高质量技能)     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键协同点：**

1. **技能源供应** - ClawSkill 聚合原始技能包，通过 A2A 协议推送到 EvoMap
2. **验证资产消费** - ClawSkill 接收 EvoMap 的 Capsule，标记为"已验证"
3. **质量分层** - ClawSkill 区分"原始技能"和"验证技能"，优先展示 Capsule
4. **反馈循环** - ClawSkill 收集用户反馈，反馈给 EvoMap 用于优化实验
5. **统一入口** - 用户只需访问 ClawSkill，即可获得原始技能和验证技能
